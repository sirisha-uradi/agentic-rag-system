# 🤖 Agentic RAG System

An intelligent Question Answering system built on the 
"Attention Is All You Need" research paper using 
Retrieval-Augmented Generation with an agentic 
self-correcting loop.

## 🎯 What This Project Does
Ask any question about AI research papers and get 
accurate intelligent answers — or an honest 
"I don't know" when the answer isn't available.

## ✨ Features
- 📄 Automatic PDF processing and chunking
- 🧮 Semantic vector search using embeddings
- 🤖 Agentic loop with self correction
- 📊 Evaluation harness with 100% accuracy score
- 🌐 Beautiful Streamlit web interface

## 🏆 Evaluation Results
- Total Questions Tested: 10
- Accuracy Score: 100%
- Relevant Questions Answered: 7/7
- Irrelevant Questions Rejected: 3/3

## 🔧 Tech Stack
- Python 3.14
- Groq API (Llama 3.1)
- ChromaDB (Vector Database)
- Sentence Transformers
- Streamlit
- PyPDF

## 🚀 How To Run
1. Clone this repository
2. Install requirements: pip install groq 
   chromadb sentence-transformers pypdf streamlit
3. Add your Groq API key in app.py
4. Run: streamlit run app.py
